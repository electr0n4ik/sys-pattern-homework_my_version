1.2. Создайте учётную запись sys_temp. 

Вход в mysql:

```shell
sudo mysql
```

Создание нового пользователя:

```shell
CREATE USER 'sys_temp'@'localhost' IDENTIFIED BY '12345';
```
Ответ: 

```shell
Query OK, 0 rows affected (0,01 sec)
```
